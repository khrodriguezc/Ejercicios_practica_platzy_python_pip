# Game Project

Para correr el juego debes seguir las siguientes instruciones en la terminal
``` sh

```

# App Project
Para ejecutar la app diseñada es necesario:

``` sh
git clone
cd app
python3 -m venv env_1
source env_1/bin/activate
pip3 install -r requirements.txt
python3 main.py
```
